El programa desarrollado tiene como objetivo analizar una matriz cuadrada de tamaño n×n, cuyos valores son generados 
de forma aleatoria (0 y 1). A partir de esta matriz, se busca identificar la fila que contiene la mayor secuencia consecutiva de unos (1).

Para lograrlo, el programa realiza los siguientes pasos:

Solicita al usuario el tamaño de la matriz.
Genera automáticamente los valores de la matriz con números binarios (0 o 1).
Imprime la matriz generada para su visualización.
Recorre cada fila evaluando la cantidad de unos consecutivos presentes.
Determina cuál es la secuencia más larga de unos dentro de toda la matriz.
Identifica la fila en la que ocurre dicha secuencia.
Muestra como resultado:
La fila donde se encontró la mayor secuencia.
La longitud de esa secuencia de unos consecutivos.

Este tipo de problema permite aplicar estructuras de control como ciclos anidados, condiciones y uso de contadores, siendo 
fundamental para el desarrollo de la lógica de programación orientada al análisis de datos en matrices.
